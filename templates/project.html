Project Details

Methodology

We rely on a curated portfolio of tabular ML pipelines—tree ensembles (LightGBM, XGBoost, CatBoost, Gradient Boosting) and a calibrated Multilayer Perceptron—to score each candidate. Every pipeline shares a consistent preprocessing stack, cross-validation protocol, and explainability reports so their outputs can be compared side by side.

Data Source

We aggregate the latest public catalogues from NASA's TESS TOI archive, the Kepler KOI list, and the extended K2 mission. Each catalogue ships with harmonised feature schemas, metadata lookups, and clean training splits to make retraining or experimentation frictionless.

Available Production Models

The web experience exposes the following pre-trained models for instant inference and comparison:

  • TESS — LightGBM, XGBoost, CatBoost, Gradient Boosting, Random Forest, MLP
  • Kepler — LightGBM, XGBoost, CatBoost, Gradient Boosting, Random Forest, MLP
